## Context
SharkEngine is a Fabric 1.21.1 mod. Prism Launcher is installed via Flatpak.
- Build dir: `sharkengine/build/libs/`
- Prism mods dir: `~/.var/app/org.prismlauncher.PrismLauncher/data/PrismLauncher/instances/1.21.1/minecraft/mods/`

## Your Task

Build the mod JAR and copy it to Prism Launcher's mods folder for in-game testing.

### Steps

1. Run `./gradlew --no-daemon build` in the `sharkengine/` directory
2. Verify build succeeded and all tests passed
3. Check the built JAR exists at `sharkengine/build/libs/sharkengine-*.jar`
4. Verify JAR contents with `jar tf` — confirm key resources are present (tags/block/, lang/, sounds/)
5. Copy JAR to Prism Launcher mods directory, replacing the existing one
6. Re-enable any `.disabled` mods in the Prism mods folder (fabric-api, controlify, yacl)
7. Report: JAR size, timestamp, and remind user to restart Minecraft in Prism Launcher

### Guardrails
- Always verify the build succeeded before copying
- Check that `fabric-api*.jar` is NOT disabled (rename `.disabled` back if needed)
- Never delete existing mods — only replace the sharkengine JAR
- The Prism instance name is "1.21.1" (may change in future)
